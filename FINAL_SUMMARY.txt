【核心功能】

1. 🔍 Find 模式（查找定位）
   ├─ 左右分栏专用布局
   ├─ 单图上传和处理
   ├─ Canvas 边界框可视化
   ├─ 6种彩色霓虹标注
   ├─ 精确坐标转换
   └─ 响应式自动重绘

2. 📦 批量处理模式
   ├─ 多图片上传
   ├─ 拖拽排序
   ├─ 逐一处理
   ├─ 实时进度跟踪
   └─ 结果自动合并

3. 🌐 多语言支持
   ├─ 🇨🇳 简体中文 (zh-CN)
   ├─ 🇹🇼 繁體中文 (zh-TW)
   ├─ 🇺🇸 English (en-US)
   ├─ 🇯🇵 日本語 (ja-JP)
   ├─ UI完整翻译
   └─ 设置自动保存

4. ⭐ GitHub 项目推广
   ├─ Header GitHub 链接
   ├─ Footer Star 大横幅
   ├─ 快捷链接（4个）
   ├─ 版权信息
   └─ 多语言支持

5. 🖼️ 图像描述           - 批量处理 | AI描述

7. ✨ 自定义 (Freeform)  - 批量处理 | 自由提示

推荐工作流:

1. 批量文档处理:
   选择 "文档转Markdown" → 上传多张 → 开始识别 → 下载结果

2. 精确查找定位:
   选择 "查找定位" → 上传单图 → 输入关键词 → 查看标注

3. 自定义识别:
   选择 "自定义提示" → 输入提示词 → 上传图片 → 获取结果
